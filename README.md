# Linear Regression From Scratch

Implemented Simple Linear Regression using NumPy without using sklearn's LinearRegression.

## Formula

m = Σ((x - x̄)(y - ȳ)) / Σ((x - x̄)^2)

b = ȳ - mx̄

## Results

Dataset: Placement Dataset

R² Score: 0.78

## Concepts Learned

- Mean Centering
- Least Squares Method
- Train/Test Split
- Prediction
- R² Evaluation
